Working on TF2 Casual: Reimagined, the future plans of two Minecraft servers

Posted by Roon on June 14, 2021

Hi everyone,

It's been while since the last Starlight Paradise post, right?
The project lead Roon had several trauma and stressful days, we had to take a break from development, since there was many incidents regarding Roon.

Also there was 1.11 SourceMod controversy we had to take a time to find the solution about that incident, here's what happened.

We are currently working on improved bot system for our project TF2 Casual: Reimagined, which map rotation changes dynamically when certain player count has reached, so they can play on non-bot supported maps.

Also after we start the project from scratch, we decided not to add gamemode tweaks so people can learn how to play TF2 easily and to prevent any future issues.

And, about the both Minecraft Project, we tested the earth type map but sadly there was serious issue regarding of size of the map. The map size simply exceeds over 25G and the server couldn't handle it so we decided to create improved world map instead.

While we had plan to use Bukkit API based plugins, but we decided to use Sponge API based plugins instead of Bukkit API. This helps keep our server much modern because of their own Modern API, and smaller plugin based since we only add feature which is necessary(So not cosmetics and such).

The reason of ditching Bukkit API based is because of outdated API and legal issue(Bukkit DMCA Takedown in the past).

Welp, that's what happened recently! We apologize that we haven't posted the news related to Starlight Paradise, hopefully we will be back with another development update.

Thank you,
Roon

Introducing new changes to Roon's L4D2 Modpack

Posted by Roon on June 8, 2021

Hi fellow survivors!

We made some new changes to Roon's L4D2 Modpack!
New brand has been applied to modpack, under new name 'Death Reborn'. New name and logo has been applied to the project.

Also Death Reborn GitHub profile is now available here.

Additionally, the update policy was changed. Now we no longer use the update date on version number, instead we will use following version scheme: YYYY.MM(A/B) 'Codename'.
Also the delay between new version will be 14 days+ instead of 7 days+, that means new version will release 2 times for each months.

This version number will be the 2021.06A 'Merciless'! Changelog can be seen here.

You can install our modpack on our Steam Workshop page!

We hope you enjoy the new modpack experience!
